O projeto desenvolvido é um jogo de perguntas e respostas matemático intitulado "Duck's Math". Este jogo foi concebido para atender especificamente ao conteúdo curricular do terceiro ano do ensino médio, oferecendo uma abordagem alternativa para o aprendizado de matemática. Com um design visual inspirado em jogos retrô, o projeto busca criar um ambiente de aprendizado que vá além do método tradicional de sala de aula, fazendo uso de tecnologias digitais para enriquecer a jornada de cada estudante.
